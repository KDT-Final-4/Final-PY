# Final PY API

FastAPI 기반의 콘텐츠 생성/발행 자동화 백엔드입니다. 트렌드 키워드 수집, 키워드 정제, 상품 연관도 평가, 홍보 문구 생성, 크롤링/업로드/발행 등 마케팅 콘텐츠 파이프라인을 API로 제공합니다.

## 실행 방법

Docker로 실행하는 것을 권장합니다.

1. `.env.example`을 참고해 `.env` 파일을 작성합니다.
2. 아래 명령으로 컨테이너를 빌드/실행합니다.

```bash
docker compose up --build
```

실행 후 기본 헬스 체크는 `GET /health`로 확인할 수 있습니다.
